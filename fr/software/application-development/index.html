§ Collaboration

Découverte → Conception → Construction → Transfert.

01 · Découverte
01
Découverte

Un partenaire principal mène le lancement : entretiens avec les utilisateurs, cartographie des parties prenantes, registre d'hypothèses. Résultat : un sommaire de produit + une cartographie des parcours.

02 · Conception
02
Conception

Architecture de l'information, maquettes filaires, système de conception et prototypes cliquables dans Figma. Révisés chaque semaine avec votre équipe; verrouillés avant le sprint 1.

03 · Construction
03
Construction

Sprints de 2 semaines. Web en Next.js ou React; mobile en React Native ou SwiftUI. Le pipeline CI livre en préproduction à chaque fusion; les mises en production sont cadencées par un train de livraison.

04 · Transfert
04
Transfert

Soumission à l'App Store / Play Store, transfert de l'hébergement Web, tableaux de bord de surveillance, guide de soutien et une stabilisation de 90 jours. Transition vers un contrat de service géré au 91e jour au besoin.

§ Pile technologique

Outils et technologies.

Web

Next.js, React, TypeScript, TanStack Query, Tailwind ou CSS pur avec jetons de conception, Storybook.

Next.js React TypeScript TanStack Query Storybook
Mobile

React Native (multiplateforme principal), SwiftUI (lorsque le natif iOS est requis), Kotlin / Jetpack Compose (lorsque le natif Android est requis), Expo pour l'itération rapide.

React Native SwiftUI Jetpack Compose Expo
État et données

Zustand, Redux Toolkit Query, Apollo, GraphQL, REST, temps réel via Socket.io ou WebSockets.

Zustand Apollo GraphQL WebSockets
Conception et AQ

Figma, audits d'accessibilité (WCAG 2.2 AA), Playwright pour les tests E2E Web, Detox pour les tests E2E mobiles, Sentry pour les erreurs d'exécution.

Figma Playwright Detox Sentry
