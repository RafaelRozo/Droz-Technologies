03
构建

2 周冲刺。Web 采用 Next.js 或 React；移动端采用 React Native 或 SwiftUI。CI 流水线在每次合并时部署到预发布；生产发布按发布列车节奏门控。

04 · Handover
04
交接

App Store / Play Store 提交、Web 托管交接、监控仪表盘、支持运行手册以及 90 天稳定期。如有需要，在第 91 天转为托管服务合同。

§ Stack

工具与技术。

Web

Next.js、React、TypeScript、TanStack Query、Tailwind 或带设计令牌的原生 CSS、Storybook。

Next.js React TypeScript TanStack Query Storybook
Mobile

React Native（主力跨平台）、SwiftUI（需要 iOS 原生时）、Kotlin / Jetpack Compose（需要 Android 原生时）、用于快速迭代的 Expo。

React Native SwiftUI Jetpack Compose Expo
State & data

Zustand、Redux Toolkit Query、Apollo、GraphQL、REST，通过 Socket.io 或 WebSockets 实现实时通信。

Zustand Apollo GraphQL WebSockets
Design & QA

Figma、无障碍审计（WCAG 2.2 AA）、用于 Web E2E 的 Playwright、用于移动端 E2E 的 Detox、用于运行时错误的 Sentry。

Figma Playwright Detox Sentry
