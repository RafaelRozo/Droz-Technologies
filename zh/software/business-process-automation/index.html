§ Capabilities

我们自动化什么。

01 · RPA

机器人流程
自动化

针对没有可用 API 的遗留系统的屏幕抓取与 UI 驱动机器人。UiPath 或 Automation Anywhere。隔离的运行时、审计日志，以及随着底层系统现代化而有计划的退役方案。

UiPath Automation Anywhere 审计日志
02 · API Integrations

系统对系统直连，
经契约测试

系统对系统的直接对接。有 API 之处用 REST 和 GraphQL；没有之处用 Webhook 桥接。经契约测试、幂等，并带有重试和死信处理。

REST GraphQL Webhook
03 · Workflow Orchestration

以代码形式
承载业务流程

以代码建模的业务流程：分支、重试、人工审批步骤、SLA，以及可重放的状态机。在工作负载有需要时采用 Temporal、Camunda 或 Argo Workflows。

Temporal Camunda Argo
04 · iPaaS

能长期维护的
平台最重要

Zapier、Make、n8n 或 Tray.ai——按工作负载选型。你能长期维护的平台，比连接器列表最长的平台更重要。

Zapier Make n8n Tray.ai
